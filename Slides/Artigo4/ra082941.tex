\section{Introdução}

\begin{frame}{Introdução}
  \begin{itemize}
  \item Atualmente, protocolos web perderam campo para web design devido à uma preocupação menos técnica e mais efetiva.
  \item A web provocou mudanças no ensinho, na indústria, na política.
  \item Há mais documentos web que pessoas no mundo.
  \item Ferramentar facilitam criação de sistemas distribuídos e a utilização de algoritmos distribuídos.  Ex: Hadoop
  \item Computação social englova vários aspectos da interação humana na web (Ex: redes sociais)
    
  \end{itemize}
  
  \pause \begin{alertblock}{Web} %iterative cycle
    
    \begin{itemize}
      \pause  \item  Para CS é um meio de comunicação, um veículo de difusão de conteúdo.
      \pause  \item  Para os novos cursos: é um mecanismo de conteúdo dinâmico que suporta interação social entre vários usuários. 

    \end{itemize}
  \end{alertblock}

\end{frame}


\begin{frame}{}
  \begin{itemize}
  \item A web precisa ser entendida e estudada como um fenômeno, mas também como algo a ser desenvolvido para um crescimento futuro e futuras capacidades. 
  \end{itemize}

  \pause \begin{alertblock}{Web} %iterative cycle
    
    \begin{itemize}
      \pause  \item  Micro escala: Web é uma infra estrutura de linguagens artificiais e protocolos.
      \pause  \item  Macro escala: A interação dos seres humanos como criar, conectar e consumir informação.
    \end{itemize}
  \end{alertblock}


\end{frame}


\begin{frame}{}
  \begin{itemize}
  \item A web não foi desenvolvida utilizando um ciclo de desenvolvimento tradicional(especificar, projetar, desenvolver) visto em engenharia de software. 
  \item O sucesso de uma tecnologia web está atrelada a interação
    social.
  \item Além da análise de questões tecnologia, questões sociais
    passam a fazer parte do escopo de desenvolvimento.
 \end{itemize}


\end{frame}

\begin{frame}{A máquina social}
  \begin{itemize}
  \item Sucesso ou não de uma tecnologia web depende muitas vezes de
    recursos sociais. Logo, criar um sistema requer o conhecimento de
    aspectos sociais.
  \item Talvez uma forma de máquina social hoje não venha a ter grande
    efeito no futuro
  \item Muitas questões surgiram:

  \begin{itemize}
\item Como diferenças culturais afetam o desenvolvimento e o uso de
  mecanismos sociais na web? 
\item Como os princípios da arquitetura fundamental são necessários
  para guiar a projeção e o desenvolvimento de novos componentes de
  infraestrutura web para os softwares sociais?   
 \end{itemize}
 \end{itemize}


\end{frame}


\section{The web data}
\begin{frame}{The Web data}
\begin{itemize}
\item Artigos, blogs, fotos, videos, e todas as formas de recurso
  podem ser anotados utilizando palavras chaves-definidas pelo usuário
  (tag). Essas anotação são utilizadas posteriormente pelos mecanismos
  de busca.
\item Tags são utilizado como metadata lara ajudar a explicar o
  conteúdo dos objetos. 
\item A web permite ao usuários e programadores trabalhar com um
  conjunto de documentos interconectados sem se preocupar com detalhes
  da localização, de como é armazenado ou como occore a transferência
  de dados (Transparência de sistemas distribuídos)
\item Um dos objetivos, hoje , é criar ferramentas que acelerem a
  inferenecia em grandes bases de conhecimento. 
\end{itemize}
\end{frame}




\section{Conclusão}
\begin{frame}{Conclusão}
\begin{itemize}
\item A web é diferente de todos os outros sistemas
  já estudados pois ocorre uma alta taxa de mudanças nele.   
\item O futuro da humanidade está ligado ao futuro da web.
\item Logo, existe o dever de que os futuros desemvolvimentos da web
  tornem um mundo um local melhor.
  
\item É necessário novas metodologias para coleta de evidências 
  dência e encontrar maneiras de antecipar 
  como o comportamento humano afeta o desenvolvimento de um sistema que está evoluindo a uma taxa tão surpreendente 
 
\end{itemize}
\end{frame}
